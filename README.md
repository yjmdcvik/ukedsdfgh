# VLESS Heroku

## 概述

用于在 Heroku 上部署 vless+websocket+tls，每次部署自动选择最新的 alpine linux 和 xray core。  
vless 相比 vmess 性能更加优秀，占用资源更少，运行更加稳定。

## 镜像

经测试本镜像占用内存资源较低，运行稳定。
